# Experiments List

### **Ex 1: Webpage Design**
Design a webpage for your college containing descriptions of courses, departments, faculties, library, etc.

### **Ex 2: CSS Styling Techniques**
Create a webpage and use inline, embedded, and external CSS. Experiment with CSS3 features like background images, border images, colors, and shadows.

### **Ex 3: Student Registration Form (JavaScript Validation)**
Create a student registration form for job applications and validate all form fields using JavaScript.

### **Ex 4: Adaptive Quiz Program**
Create a quiz program with adaptive questions along with statistical variations using JavaScript.

### **Ex 5: JSP Registration Insert**
Design a JSP page to insert the details of users who register through the registration form and store the details into the database.

### **Ex 6: JavaScript Event Handling**
Implement various event-handling features in web pages.

### **Ex 7: API Widget**
Create a simple widget (weather or stock ticker) using JSON data fetched from an API.

### **Ex 8: JDBC Bank Transactions**
Implement JDBC connectivity to retrieve bank customers' transaction details.

### **Ex 9: PHP Registration Validation**
Develop a PHP program to validate the fields of the Registration page.
